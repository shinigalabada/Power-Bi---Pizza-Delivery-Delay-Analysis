# Pizza Delivery Performance & Delay Analysis Dashboard

## Project Summary

A data analytics project built using a real-world American pizza delivery dataset (**1,005 records, 25 variables**) to uncover performance trends and identify key drivers of delivery delays across five major pizza chains:

-  Domino’s
-  Little Caesars
-  Papa John’s
-  Pizza Hut
-  Marco’s Pizza

Developed as a group project using **Power BI**, focusing on interactive visual storytelling and operational insights for business decision-making.

---

##  Dashboard Overview

The solution consists of an interactive Power BI dashboard with three key pages:

- **Home** 
- **Restaurant Overview** – Comparative analysis of restaurant performance
- **Delay Analysis** – Deep dive into delivery delay patterns and root causes

---

##  Interactive Capabilities

- Peak hour/day analysis filters
- Month and location slicers
- One-click reset for exploration flexibility

---

##  Key Performance Indicators (KPIs)

- Total Orders
- Average Delivery Time
- On-Time Delivery Rate
- Average Delay Time

---

---

##  Key Insights & Business Impact

-  Overall delivery delay rate: **21%**
-  Peak hours significantly increased delivery delays
-  High traffic conditions were the strongest driver of delay time
-  Complex pizza orders contributed to longer delivery times
-  Delivery distance showed a moderate positive correlation with delays
-  Slight variation observed between weekday and weekend delays
-  Domino’s recorded the highest order volume among all chains
